# down

### install with curl

```bash
curl raw.github.com/jeromedecoster/down/master/install.sh \
    --location \
    --silent \
    | bash
```

### install with wget

```bash
wget raw.github.com/jeromedecoster/down/master/install.sh \
    --output-document=- \
    --quiet \
    | bash
```

## Notes

Login / password called from `~/.netrc` file
